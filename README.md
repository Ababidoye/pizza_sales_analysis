# pizza_sales_analysis
The Project analyzes a dataset consisting of four csv files for pizza and pizza orders and another csv file that gives additional information on the dataset.The dataset were merged using python pandas library. Thereafter further modification (addition and dropping of columns) was done on the merged dataframe to get a clean dataframe for analysis.
Questions answered in this project include: The total pizza orders made, total quantity of pizza ordered, quantity of pizza ordered and revenue generated by category, total revenue made, analysis of monthly, daily and hourly sales, ingredient usage rate and so on.
Visualization was also done using matplotlib and seaborn
